* Documents Utils

** Usage

#+BEGIN_SRC bash
  docker run --rm -p 5000:5000 chanmo/doc-utils
#+END_SRC

** APIs

*** Format Convert

use httpie, convert to pdf
#+BEGIN_SRC bash
  http POST -f :5000/convert_to/pdf file@~/demo.docx
#+END_SRC


use httpie, convert to html
#+BEGIN_SRC bash
  http POST -f :5000/convert_to/html file@~/demo.docx
#+END_SRC
